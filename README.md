# Projeto de Order Client

Esse projeto tem o objetivo de resolver o problema de cadastramento de clientes e pedidos de produtos
usando Classes, Listas, Combinações e Enums. 

O sistema faz o cadastro de clientes, observa o situação do pedido, faz o pedido com base em quantos
produtos o cliente deseja e no final mostra um resumo do pedido.

Nesse projeto, eu consegui entender a questão de organização (Entities), a numerar os status dos pedidos,
a construir uma lista de forma mais precisa e casando com as outras classes.
